# 🌾 Mandi Mitra

MandiMitra is a Gemma-powered advisory tool that combines weather forecasts and mandi prices to help farmers make informed crop-selling decisions.

![Architecture](assets/architecture.png)

## 📖 Overview

**Mandi Mitra** is an AI-powered decision support system that helps Indian farmers make informed selling and harvesting decisions by combining **real-time mandi prices**, **weather forecasts**, and **crop-specific agricultural knowledge**. Instead of simply displaying raw data, the system analyzes multiple factors using a rule-based decision engine and generates practical, localized advisories in regional languages through **Google Gemma**.

The project is designed to provide farmers with timely recommendations while ensuring that advice remains factual, explainable, and cautious. It does **not** predict future prices or guarantee profits; rather, it presents current market conditions and weather risks to support better decision-making. :contentReference[oaicite:0]{index=0}


# 🚀 Motivation

Small and marginal farmers often face uncertainty when deciding **when to harvest, transport, or sell their produce**. Market prices fluctuate daily, while unexpected weather conditions can significantly impact crop quality and transportation.

Mandi Mitra addresses this challenge by integrating:

- 📈 Live AGMARKNET mandi prices
- 🌦 Weather forecasts
- 🌱 Crop-specific storage and sensitivity information
- 🤖 Gemma-powered multilingual advisory generation

into a single easy-to-use application.

# 🏗 System Architecture

The overall workflow of Mandi Mitra is illustrated below.

![Architecture](assets/architecture.png)

### Workflow

1. Farmer selects:
   - State/District
   - Crop
   - Preferred language

2. Backend retrieves:
   - Latest mandi prices from AGMARKNET
   - Historical price records
   - Weather forecast
   - Crop knowledge

3. Intermediate Rule Layer:
   - Computes 7-day average price
   - Determines market trend
   - Detects weather risks
   - Evaluates crop constraints

4. Structured facts are passed to Gemma.

5. Gemma generates an easy-to-understand advisory in the selected language.

6. The application displays:
   - Current market price
   - Price trend
   - Weather summary
   - Recommended action
   - Supporting evidence
   - Disclaimer

# ⚙️ How It Works

## 1. Farmer Input

The farmer selects:

- State
- District
- Crop
- Preferred language


## 2. Market Data Collection

The system retrieves:

- Current mandi price
- Historical market prices
- Modal price
- Market location

using the AGMARKNET API.


## 3. Weather Collection

A 5-day weather forecast is fetched using the OpenWeather API, including:

- Temperature
- Rain probability
- Weather conditions



## 4. Crop Knowledge

Each supported crop has predefined agricultural information, including:

- Optimal temperature range
- Critical temperature thresholds
- Storage life
- Harvest window
- Perishability
- Rain sensitivity
- Heat sensitivity
- Grading standards


## 5. Rule-Based Decision Engine

Instead of allowing the language model to interpret raw numerical values directly, Python first analyzes the collected information to derive structured facts.

The rule engine evaluates:

- Seven-day average price
- Price trend (Rising / Falling / Stable)
- Heat risk
- Cold risk
- Rain risk
- Crop perishability
- Storage duration

Based on these factors, it generates one of the following recommendations:

- Sell within 1–3 days
- Sell soon
- Monitor prices before selling


## 6. AI Advisory Generation

The structured decision generated by the rule engine is provided to **Gemma**, which converts it into concise, farmer-friendly advice in the selected regional language.

The generated advisory includes:

- Current situation
- Weather impact
- Suggested action
- Important caution

# 🔒 Safety & Limitations

Mandi Mitra is designed as a **decision-support system** and not a predictive trading platform.

The application:

- Does **not** predict future prices.
- Does **not** guarantee profits.
- Does **not** recommend pesticides or fertilizers.
- Generates advice only from verified market, weather, and crop information.

Farmers are encouraged to verify local market conditions before making final decisions.
